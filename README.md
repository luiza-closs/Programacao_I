# 📘 Programação I

Repositório destinado aos **projetos, exercícios e trabalhos avaliativos** da disciplina de **Programação I**, cursada no **2º semestre de Ciência da Computação**.

O conteúdo aborda principalmente os **fundamentos de Programação Orientada a Objetos (POO)**, incluindo modelagem de classes, encapsulamento, herança, polimorfismo e interfaces, com exemplos práticos desenvolvidos em Java.

---

## 🎯 Objetivos da Disciplina

Ao longo deste repositório, são trabalhados conceitos como:

- Estruturação de classes e objetos
- Atributos e métodos
- Encapsulamento (`getters` / `setters`)
- Validação de regras de negócio
- Relacionamento entre classes
- Herança
- Polimorfismo
- Interfaces e contratos
- Introdução a interfaces gráficas (`JFrame`)
